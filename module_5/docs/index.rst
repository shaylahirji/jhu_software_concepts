Project Overview
----------------
The Grad Cafe Application is an automated tool designed to scrape, clean, and 
analyze graduate school admission data. By leveraging a multi-layered 
architecture—including a Flask web interface and a PostgreSQL database—it 
provides users with actionable insights into admission trends.
